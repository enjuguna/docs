## Models Tab

Purpose: Manage inverter models and their register settings.

### Features

- List/select models, each is a separate CSV file
- View all registers for selected model: SlaveID, Type, Address, DataType, Scale, ByteOrder, WordOrder, Units, Desc, Expected
- Add/edit/delete registers via modal dialog (all fields validated)
- Create or delete models

### API Endpoints Used

- `GET /api/v1/models`
- `POST /api/v1/models`
- `PUT /api/v1/models`
- `DELETE /api/v1/models`
- `POST /api/v1/models/{name}/registers`
- `PUT /api/v1/models/{name}/registers`
- `DELETE /api/v1/models/{name}/registers`

### Data Stored In

- [`models/{modelname}.csv`](models/)

### UI Components

- List of models, table of registers, dialog for register management

---

### Step-by-Step Guide

1. **Viewing Models**
   - The left panel lists all models (from `models/` CSVs).
   - Select a model to view its register settings in the table.

2. **Adding a Model**
   - Enter the new model name (must end with `.csv`).
   - Click **Add Model**.
   - The new model appears in the list and is created in [`models/`](models/).

3. **Deleting a Model**
   - Select a model from the list.
   - Click **Delete Model**.

4. **Viewing and Editing Registers**
   - The table displays register settings for the selected model.
   - Columns include: SlaveID, RegisterType, RegisterAddress, DataType, Scale, ByteOrder, WordOrder, Units, Description, ExpectedValue.

5. **Adding a Register**
   - Click **Add Register**.
   - Fill in required fields.
   - Click **OK** to save.

6. **Modifying a Register**
   - Select a row in the settings table.
   - Click **Modify Register**.
   - Edit fields and click **OK**.

7. **Deleting a Register**
   - Select a row in the settings table.
   - Click **Delete Register**.
