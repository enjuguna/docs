## Inverters Tab

Purpose: Add and manage inverter devices.

### Features

- View/list all inverters (ID, Name, IP address, Port, Model, One-Based Addressing)
- Search/filter by name/IP
- Pagination (10 results per page)
- Add new inverter (fill out form, select model from dropdown)
- Click row to select inverter for editing/modification
- Edit/delete inverter (UI reflects changes instantly)

### API Endpoints Used

- `GET /api/v1/inverters`
- `POST /api/v1/inverters`
- `PUT /api/v1/inverters`
- `DELETE /api/v1/inverters`

### Data Stored In

- [`inverters/inverters.csv`](inverters/inverters.csv:1)

### UI Components

- Table with colored badges for model/one-based
- Right-hand form for edits

---

### Step-by-Step Guide

1. **Viewing Inverters**
   - All inverters are listed in a table.
   - Columns:
     - **ID:** Unique identifier (auto-generated).
     - **Inverter Name:** Descriptive name.
     - **IP Address:** Network address (e.g., `192.168.100.1`).
     - **Port:** Communication port (e.g., `502`).
     - **Model:** Select from available models (e.g., `SG4400`).
     - **One Based Addressing:** `true` or `false`.

2. **Adding an Inverter**
   - Fill in the IP Address, Port, Model, Addressing Type.
   - Click **Add**.
   - The new inverter appears in the table and is saved to [`inverters/inverters.csv`](inverters/inverters.csv:1).

3. **Editing an Inverter**
   - Select a row in the table.
   - The form is populated with the inverter's details.
   - Modify fields as needed.
   - Click **Modify** to save changes.

4. **Deleting an Inverter**
   - Select row in the table.
   - Click **Delete** to remove inverter.

5. **Sorting and Filtering**
   - Click column headers to sort.
   - Use the search/filter features (if available) to find specific inverters.
